How to Setup
Streamlabs in Mobile

Download & Install Streamlabs
on Mobile from the playstore

Download Streamlabs from Here

Open Streamlabs and
go to editor

Add layer of phone
screen to streamlabs and
stretch it to fill the screen

Go to settings on your
Goblitz.tv profile and get your
channel's RTMP URL and KEY.

Enter the same name in
Streamlabs App - > Settings - >
Streaming Platform - > Custom RTMP
Server

How to Setup
Streaming on Goblitz

Download
Streamlabs OBS

Download OBS from Here
Open and go to Streamlabs settings.

Select Streaming Platform
and select Custom Server

Go to Goblitz settings page on the Goblitz website

Copy URL and stream key from Goblitz website into
Streamlabs custom server section

Go Live on OBS

Press the red button in streamlabs
to start the Stream

View your stream

@goblitz.tv/yourname/channel

Increase your views by sharing url
of the stream on different Social platforms
